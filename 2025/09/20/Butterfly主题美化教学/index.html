注意事项

  1. 修改前备份配置文件，以便出现问题时恢复
  2. 每次修改后都使用npm run server本地预览效果
  3. 不要直接修改主题目录中的文件，这样会影响后续主题升级
  4. 使用独立的主题配置文件_config.butterfly.yml进行配置
  5. 对于复杂的布局修改，可以使用覆盖布局文件的方式
