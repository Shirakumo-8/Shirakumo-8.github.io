Markdown 语法小贴士

以下是一些常用的 Markdown 语法示例：

标题

使用 # 符号表示标题级别：

  • 一级标题

  • 二级标题

  • 三级标题

文本格式

  • 粗体文本: **粗体文本**
  • 斜体文本: *斜体文本*
  • 删除线: ~~删除线~~

列表

无序列表使用 - 或 *：

  • 项目1
  • 项目2
    • 子项目A
    • 子项目B

有序列表直接使用数字：

  1. 第一步
  2. 第二步
  3. 第三步
